PGP Implementation


An implementation of the Pretty Good Privacy (PGP) protocol for sending emails securely, ensuring Confidentiality, Integrity, and Authenticity. It allows users to encrypt, decrypt, sign, and verify emails with cryptographic keys.

Secure PGP Email Application

This application enables secure email operations using PGP encryption and signatures. Users can securely exchange emails using encryption, message signing, and authentication mechanisms.

  • Encrypt Messages: Secure your messages by encrypting them with the recipient's public key.
  • Decrypt Messages: Retrieve original messages using your private key.
  • Sign Messages: Authenticate messages with a digital signature.
  • Send Secure Emails: Combine encryption and signing for secure communication.



Fourier Transform & Signal Processing


This project focuses on the **Fourier Transform** and its applications in signal processing. The Fourier Transform allows the decomposition of signals into their frequency components, making it a fundamental tool in various fields, including audio processing, communications, and image analysis.

Real-time equalizer visualization using Fourier Transform.

Visualization of phasor combinations in signal processing.

Fourier Transform applied to an image to analyze frequency components.

  • Signal Decomposition: Breaking down complex signals into simpler sinusoidal components.
  • Noise Filtering: Enhancing signal clarity by removing unwanted frequency components.
  • Compression Techniques: Reducing data size while preserving essential information.
  • Real-World Applications: Used in audio equalization, image enhancement, and medical imaging.
